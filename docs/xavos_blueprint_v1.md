# Xavos — Blueprint v1

## 1. Visión

Xavos es mi programa local para crear y mantener:
- apps mobile
- programas de PC
- y más adelante páginas web

Todo basado en:
- plantillas propias
- reglas de calidad
- herramientas que yo realmente domino.

## 2. Objetivo

Que al presionar **“New Project”**:
- aparezca un **wizard** con preguntas claras
- pueda elegir:
  - app pre-hecha (plantilla)
  - app personalizada
- el resultado siempre sea un proyecto organizado y premium.

## 3. Piezas principales del programa

### 3.1 Wizard de proyectos
- Flow de preguntas clave (tipo de proyecto, stack, módulos).
- Opción de usar plantillas ya hechas.
- Opción de proyecto personalizado.
- Nada se genera sin confirmación final.

### 3.2 Editor integrado (sencillo)
- Editor de código básico dentro de Xavos.
- No reemplaza VSCode/Android Studio, solo lo esencial.
- Permite ver y ajustar archivos rápidos sin salir del programa.

### 3.3 Modo visual tipo “Figma”
- Vista de pantalla en modo visual (no solo código).
- Lista de widgets disponibles.
- Panel de propiedades (ejemplo: scroll, padding, opciones del scaffold).
- Si confirmo los cambios visuales → Xavos actualiza el código real.

### 3.4 Theme y Design System propios
- DS y theme base definidos por nosotros.
- El usuario puede personalizar (ej: cambiar tono de verde).
- Siempre dentro de nuestras reglas:
  - nada de colores locos fuera del sistema
  - nada que rompa la organización.

### 3.5 Biblioteca de widgets y templates
- Widgets base premium (AppScaffold, botones, listas, etc.).
- Pantallas tipo (auth, dashboard, lista, detalle).
- Módulos listos para usar más adelante (auth, gps, scan, docs, notificaciones).
- En modo visual se pueden colocar widgets y pantallas, y el código se mantiene limpio.

## 4. Filosofía

- Todo bajo reglas claras.
- Nada “mágico” fuera de control.
- Siempre organización primero, visual después.
- Xavos decide basado en lo que yo domino y en las herramientas que tengo instaladas.
