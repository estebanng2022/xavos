# Xavos — Wizard Flow v1

## Objetivo
El Wizard crea proyectos nuevos a partir de un blueprint y preguntas simples.

## Pasos del Wizard (versión inicial)

1. Nombre del proyecto
2. Tipo de app:
   - Flutter
   - Android nativo
3. Objetivo principal de la app (1 frase)
4. Complejidad:
   - Básica
   - Media
   - Avanzada
5. Módulos a incluir:
   - Auth
   - GPS
   - Scan
   - Docs
   - Notificaciones
6. Design System:
   - Usar DS por defecto
   - Sin DS (solo estructura)
7. Confirmación final:
   - Mostrar resumen del blueprint
   - Preguntar: “¿Crear proyecto? (sí/no)”

El Wizard solo crea la estructura si el usuario confirma.
