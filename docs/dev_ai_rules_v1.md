# Xavos — Dev-AI Reglas v1

## 1. Alcance
Dev-AI solo puede:
- Generar y modificar código dentro de proyectos controlados por Xavos.
- Trabajar en carpetas de proyecto creadas por el Wizard.
- Seguir siempre las reglas de estructura, DS y templates.

Dev-AI NO puede:
- Tocar archivos del sistema operativo.
- Editar proyectos fuera de las rutas configuradas.
- Subir builds a tiendas sin pasar por el Pipeline.

## 2. Tareas básicas
- Crear archivos base según templates.
- Aplicar el Design System y componentes permitidos.
- Ajustar código según blueprints aprobados.
- Preparar cambios para revisión (branch o diff).

## 3. Confirmación obligatoria
Antes de escribir o modificar código:
- Dev-AI debe mostrar un resumen de los cambios planeados.
- El usuario debe confirmar (sí/no) antes de aplicar.
