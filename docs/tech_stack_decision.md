# Xavos — Tech Stack Decision

## Core de Xavos
- Lenguaje: C#
- Plataforma: .NET
- Tipo: Servicio / app de escritorio en Windows

Motivo:
- Máximo control de la PC (procesos, archivos, hardware).
- Integración sencilla con herramientas de desarrollo (Flutter, Android, Git, etc.).
- Estabilidad y soporte enterprise.

## UI de Xavos (futuro)
- Tecnología: Flutter
- Rol:
  - Dashboard visual
  - Panel de control para proyectos, pipelines y AIs

## Apps generadas por Xavos
- Flutter → apps ligeras / medianas.
- Android nativo → apps avanzadas, muy personalizadas o con integraciones profundas.
