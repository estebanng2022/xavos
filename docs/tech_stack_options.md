# Xavos — Tech Stack Options

Aún no está decidido el stack final. Opciones posibles:

## Opción 1 — .NET (recomendado para app de escritorio)
- Lenguaje: C#
- Tipo de app: Desktop (WinUI/WPF)
- Ventajas:
  - Buen soporte en Windows
  - Fácil integrar scripts y procesos externos
  - Ideal para controlar la PC y herramientas de desarrollo
- Uso típico:
  - Interfaz principal de Xavos
  - Panel para ver estados, logs, proyectos

## Opción 2 — Node.js + CLI (modo consola)
- Lenguaje: TypeScript/JavaScript
- Tipo: Herramienta CLI
- Ventajas:
  - Simple para automatizar tareas
  - Fácil de ejecutar scripts
- Uso típico:
  - Comandos de automatización
  - Pipelines sin interfaz gráfica
