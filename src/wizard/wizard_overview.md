# Xavos — Wizard Overview

El Wizard es el sistema que crea nuevos proyectos.

## Funciones del Wizard
- Leer blueprint inicial (pantallas, módulos, complejidad).
- Decidir stack sugerido (Flutter o Nativo).
- Preguntar confirmación final al usuario.
- Crear estructura de carpetas premium.
- Importar Design System (si aplica).
- Agregar templates base (auth, gps, scan, etc.).
- Crear README del proyecto generado.

## Regla principal
El Wizard nunca genera código sin aprobación del usuario.
